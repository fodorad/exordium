.. py:class:: OpensmileWrapper(feature_set: str = 'egemaps', feature_level: str = 'lld')

   Extract acoustic features using OpenSmile.

   Wrapper for OpenSmile feature extraction with support for multiple
   predefined feature sets and levels.

   OpenSmile wrapper class.

   :param feature_set: Predefined feature set. Supported values are 'egemaps' and 'compare'.
   :param feature_level: Predefined feature level. Supported values are 'lld' and 'functionals'.

   :raises ValueError: If feature_set or feature_level is invalid.


   .. py:method:: audio_to_feature(audio_path: pathlib.Path | str, **_kwargs) -> numpy.ndarray

      Extract OpenSmile features from audio file.

      :param audio_path: Path to audio file.
      :param \*\*kwargs: Additional arguments (unused, for decorator compatibility).

      :returns: Feature array from OpenSmile processing.
