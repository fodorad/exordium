.. py:function:: load_audio(audio_path: pathlib.Path | str, target_sample_rate: int | None = 16000, clamp: bool = True, mono: bool = True, squeeze: bool = True) -> tuple[torch.Tensor, int]

   Load audio file with optional resampling and preprocessing.

   :param audio_path: Path to audio file.
   :param target_sample_rate: Target sample rate in Hz. If None, keeps original sample rate.
   :param clamp: If True, clamp waveform values to [-1, 1].
   :param mono: If True and audio is stereo, convert to mono by averaging channels.
   :param squeeze: If True, remove singleton dimensions from output.

   :returns:

                 - waveform: Audio tensor of shape (T,) if mono and squeezed,
                            (C, T) if stereo or not squeezed.
                 - sample_rate: Sample rate in Hz.
   :rtype: Tuple of (waveform, sample_rate)


.. py:function:: split_audio(audio: torch.Tensor, segment_duration: float, sample_rate: int) -> list[torch.Tensor]

   Split audio into fixed-duration segments.

   :param audio: Audio tensor of shape (T,) or (C, T).
   :param segment_duration: Duration of each segment in seconds.
   :param sample_rate: Sample rate in Hz.

   :returns: List of audio segments. Last segment may be shorter than segment_duration.
