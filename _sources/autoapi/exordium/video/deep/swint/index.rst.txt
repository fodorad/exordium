.. py:class:: SwinTWrapper(arch: str = 'tiny', pretrained: bool = True, device_id: int | None = None)

   Bases: :py:obj:`exordium.video.deep.base.VisualModelWrapper`


   Swin Transformer feature extractor wrapper.

   Loads a pretrained Swin Transformer backbone and extracts global
   average-pooled patch features, producing a ``(B, D)`` array.

   Feature dimensions by architecture:

   * ``"tiny"`` / ``"small"``: 768-d
   * ``"base"``: 1024-d

   :param arch: One of ``"tiny"``, ``"small"``, ``"base"``.
                Defaults to ``"tiny"``.
   :param pretrained: Load ImageNet-pretrained weights. Defaults to ``True``.
   :param device_id: GPU device index. ``None`` or negative uses CPU.


   .. py:method:: preprocess(frames) -> torch.Tensor

      Resize, centre-crop and normalise to ImageNet stats.

      :param frames: Any input supported by
                     :meth:`~exordium.video.deep.base.VisualModelWrapper._to_uint8_tensor`.

      :returns: Float tensor of shape ``(B, 3, 224, 224)`` on ``self.device``.



   .. py:method:: inference(tensor: torch.Tensor) -> torch.Tensor

      Swin Transformer forward pass with global average pooling.

      The backbone ``forward`` returns patch tokens of shape
      ``(B, N, C)``. This method averages over the patch dimension
      to produce ``(B, C)`` embeddings.

      :param tensor: Preprocessed tensor of shape ``(B, 3, 224, 224)``
                     on ``self.device``.

      :returns: Feature tensor of shape ``(B, D)``.
