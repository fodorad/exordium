.. py:class:: FabNetWrapper(device_id: int | None = None)

   Bases: :py:obj:`exordium.video.deep.base.VisualModelWrapper`


   FAb-Net face encoder wrapper.

   Produces a 256-dimensional identity-preserving feature vector from a face
   crop.  Model weights are downloaded automatically on first use.

   :param device_id: GPU device index. ``None`` or negative uses CPU.


   .. py:method:: preprocess(frames) -> torch.Tensor

      Resize RGB frames to 256×256 and scale to [0, 1].

      FAb-Net was trained without channel-wise mean/std normalisation.

      :param frames: Any input supported by
                     :meth:`~exordium.video.deep.base.VisualModelWrapper._to_uint8_tensor`.

      :returns: Float tensor of shape ``(B, 3, 256, 256)`` on ``self.device``.



   .. py:method:: inference(tensor: torch.Tensor) -> torch.Tensor

      FAb-Net encoder forward pass.

      :param tensor: Preprocessed face tensor of shape (B, 3, 256, 256)
                     on self.device with values in [0, 1].

      :returns: Identity feature tensor of shape (B, 256).
