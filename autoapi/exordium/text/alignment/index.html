exordium.text.alignment¶

Word-level forced alignment and open-vocabulary word timestamps.

Two public classes:

  • TorchaudioForcedAligner — aligns a known transcript to audio using torchaudio’s multilingual MMS_FA bundle. No extra dependencies beyond the pinned torchaudio.

  • WhisperWordTimestamper — transcribes raw audio with WhisperWrapper and then forced-aligns the result, yielding open-vocabulary word timestamps from audio alone.

Both return list[Word] (see Word), the common currency consumed by exordium.text.transcript_align.

exordium.text.alignment.normalize_words(transcript: str) → list[str][source]¶

Normalize a transcript into aligner-friendly lowercase word tokens.

Lowercases, strips diacritics (NFKD), and removes every character outside [a-z'] so the tokens match the MMS_FA dictionary. Empty tokens are dropped. Used by TorchaudioForcedAligner.

Parameters:

transcript – Raw transcript text.

Returns:

List of normalized word tokens (possibly empty).

class exordium.text.alignment.TorchaudioForcedAligner(device_id: int | None = 0, pretrained: bool = True)[source]¶

Bases: exordium.text.base.ForcedAligner

Forced aligner backed by torchaudio.pipelines.MMS_FA.

MMS_FA is a Massively Multilingual Speech CTC alignment model covering 1000+ languages; it produces accurate word timings by aligning the known transcript against the audio (the same idea whisperX uses, but with a single multilingual model and no extra dependencies).

Parameters:

device_id – Device index. -1 or None → CPU, 0+ → GPU/MPS.

Example:

aligner = TorchaudioForcedAligner(device_id=None)
words = aligner.align("audio.wav", "hey guys what do you want to eat")
for w in words:
    print(w.text, w.start, w.end, w.score)
min_align_samples(text: str) → int[source]¶

Samples MMS_FA needs to align text, counted with its own tokenizer.

Overrides the base class’s character estimate: normalize_words strips everything outside the MMS_FA alphabet, so counting raw characters would demand more audio than the model actually needs.

Parameters:

text – The text to be aligned.

Returns:

Minimum slice length in samples.

align(audio: pathlib.Path | str | numpy.ndarray | torch.Tensor, transcript: str, language: str | None = None) → list[exordium.text.base.Word][source]¶

Align transcript to audio and return timed Word objects.

Parameters:

  • audio – Audio file path, numpy array, or torch tensor (16 kHz mono).

  • transcript – The known transcript for this audio.

  • language – Unused — MMS_FA is multilingual. Accepted for interface compatibility with ForcedAligner.

Returns:

Words in chronological order. Empty if transcript has no alignable tokens, or if the audio is too short to carry them — see wav2vec2_min_samples(). Callers aligning whole recordings should go through align_segments(), which widens short windows instead of giving up on them.

class exordium.text.alignment.WhisperWordTimestamper(whisper: WhisperWrapper | None = None, aligner: exordium.text.base.ForcedAligner | None = None, device_id: int | None = 0, pretrained: bool = True)[source]¶

Bases: exordium.text.base.WordTimestamper

Open-vocabulary word timestamps: Whisper ASR + forced alignment.

Transcribes raw audio with WhisperWrapper, then forced-aligns the transcript with a pluggable ForcedAligner (MMS_FA by default, or WhisperxForcedAligner). This mirrors the whisperX architecture (ASR → wav2vec2 alignment) while running on the repo’s pinned torch stack.

Parameters:

  • whisper – A ready WhisperWrapper; created with device_id if omitted.

  • aligner – A ready ForcedAligner; a TorchaudioForcedAligner is created with device_id if omitted.

  • device_id – Device used only when whisper / aligner are not supplied.

Example:

ts = WhisperWordTimestamper(device_id=None)
words = ts.transcribe_words("audio.wav")
